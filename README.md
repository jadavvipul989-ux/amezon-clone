# amezon-clone
Amazon E-Commerce Clone | HTML5, CSS3, JavaScript
I built a high-fidelity Amazon clone to master the core technologies of modern web development, focusing on creating a seamless and responsive user experience. By structuring the project with clean, semantic HTML5 and advanced CSS Grid and Flexbox, I successfully replicated a complex e-commerce layout that looks great on any device. I brought the site to life using vanilla JavaScript to handle real-time DOM updates, such as a functional shopping cart counter and an interactive search bar. This project allowed me to bridge the gap between static design and dynamic functionality, ensuring fast load times and a polished, professional feel that mirrors a real-world shopping platform.
